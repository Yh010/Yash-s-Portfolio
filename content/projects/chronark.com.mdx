IngressCraft simplifies Kubernetes Dashboard accessibility by configuring a custom Ingress resource. This project, represented by the 'dashboard-ingress.yaml' file, facilitates smooth web access to the Kubernetes Dashboard at 'dashboard.com'. The Ingress rules are defined to efficiently route traffic to the 'kubernetes-dashboard' service on port 80. IngressCraft streamlines the setup, enhancing the usability and accessibility of the Kubernetes Dashboard within the specified namespace
